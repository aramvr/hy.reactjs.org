### Ինչպե՞ս կարող եք մեկնաբանություններ գրել JSX-ում {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* Մեկնաբանություն */}
  Բարև, {name}
</div>
```

```jsx
<div>
  {/* Կարող եք նաև այսպես
  բազմատող մեկնաբանությունների համար */}
  Բարև, {name}
</div>
```
