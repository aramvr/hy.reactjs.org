### Ինչպե՞ս կարող եմ մեկնաբանություններ գրել JSX-ում {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* Սա մեկնաբանություն է */}
  Բարև, {name}
</div>
```

```jsx
<div>
  {/* Սա բազմատող
 մեկնաբաություն է */}
  Բարև, {name}
</div>
```
